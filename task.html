Решение алгоритмической задачи

Задача: Можно ли из бревна, имеющего диаметр поперечного сечения D, выпилить квадратный брус шириной A?

Решение: Для решения задачи необходимо сравнить диагональ квадратного бруса с диаметром бревна. Если диагональ квадрата меньше или равна диаметру бревна, то брус можно выпилить.

Диагональ квадрата вычисляется по формуле: d = A × √2
